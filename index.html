FlexBox Row

Use these three porperties to create a Flexbox row layout

.row {
display: flex;
flex-direction: row;
flex-wrap: wrap;
}
Flexbox Column

Use this to create a flexbox column layout

.column{
display: flex
flex-direction: column
}

Box Transition Glow

Use transition and box shadows to glow on hover

.code-card .card-header {
border-radius: 8px;
transition: all 0.5s ease-in-out;
}
.code-card:hover,
.code-card:hover .card-header {
box-shadow: inset 0px 0px 8px rgba(232, 102, 236, 1),
0 0 15px rgba(232, 102, 236, 1);
}
